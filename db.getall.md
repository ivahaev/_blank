# $db.GetAll
Метод `$db.GetAll()` возвращает все объекты из хранилища. 

## Синтаксис  

```
$db.GetAll([store])
```  

## Параметры

### store  
Идентификатор хранилища (строка, опционально). По умолчанию используется текущее хранилище. 

## Возвращаемые значения  
Массив объектов, или `null` в случае ошибки.
